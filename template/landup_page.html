Xamlysis:An Online Examination Portal

Xamlysis is an online platform for students to test out there skills and analyze their performance and also, an examcraft site for teachers to set up questions for their class students in order to monitor their progress.

A proper class system with identifications for student and teachers to watch over their courses and exams in an individualized interface.
An interactive test system to attempt tests setup by the teachers for their students with a real time based system, with actual examination styles.

Features

PERFORMANCE ANALYSIS

Students would be able analyze their performance after tests with their scores along with relative performance which would help them to improve for their final goals.

EASY TO SET - UP THE EXAMINATION PAPER

With the inbuilt question set up interface, it is easier to set up and adjust all examination queries like questions, timings, assigning marks, duration of exam, etc. .

IMPROVE ACCURACY AND TIME MANAGEMENT

Students will be able to improve their accuracy with actual exam patterns and also would have better time management with the real-time based examination pattern.
